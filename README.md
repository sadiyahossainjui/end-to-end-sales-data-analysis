# End-to-End Sales Data Analysis Project

## 📌 Project Overview
This project demonstrates a complete data analysis workflow using Python, SQL, and Excel.

## 🔄 Workflow
1. Cleaned raw sales data using Python and pandas
2. Imported cleaned data into SQL
3. Performed business analysis using SQL queries
4. Built an Excel dashboard to visualize insights

## 🛠️ Tools Used
- Python
- pandas
- SQL / MySQL
- Microsoft Excel
- Pivot Tables
- Charts

## 📊 Key Insights
- Ottawa generated the highest revenue
- Electronics was the top-performing category
- Laptop generated the highest product revenue

## 🚀 Skills Demonstrated
- Data cleaning
- SQL analysis
- Data visualization
- Dashboard creation
- Business insight generation
